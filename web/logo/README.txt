MAYATRAIL — LOGO FILES
======================

  mayatrail-lockup-constellation-dark.svg   — for dark backgrounds
  mayatrail-lockup-constellation-light.svg  — for light backgrounds
  mayatrail-lockup-triangle-dark.svg        — for dark backgrounds
  mayatrail-lockup-triangle-light.svg       — for light backgrounds

Mark only (no wordmark, pure vector):
  mark-constellation.svg
  mark-triangle.svg

Notes
-----
- Colors: cool #6aa8ff, hot (breach node) #ff4b6e, deep blue (light bg) #2f5fa8.
- The wordmark uses a system sans stack (weight 600). For pixel-identical
  rendering everywhere, open in Illustrator/Figma and OUTLINE the text
  (Type > Create Outlines). The mark itself is already pure vector.
- "Trail" carries a left-to-right opacity fade (gentle) + 3 trailing dots.
- viewBox is transparent — drop on any background.
